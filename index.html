Proyectos

Actividad de cierre Profesionales tramo 1 – Línea Sarmiento

Este proyecto es una página web diseñada para simular posibles mejoras del sistema interno actual de GPS de Trenes Argentinos, específicamente para la Línea Sarmiento, Ramal Once-Moreno. La página muestra información en tiempo real sobre movimientos de trenes, estado de las vías, trabajos programados y notificaciones de imprevistos operativos.

Aplicación Mobile Club Deportivo

Este proyecto fue diseñado con mis compañeros Cristian Descosido y Antonio Gill para la materia de Desarrollo de Aplicaciones para Dispositivos Moviles de la Tecnicatura en Desarrollo en Software del IFST 29. Al hacer clic en la imagen se puede visualizar una animación del flujo completo.
